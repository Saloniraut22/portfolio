# Saloni Raut Portfolio

React + Vite + Tailwind portfolio focused on Product/UX case studies.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment (GitHub Pages)

This repo deploys automatically via `.github/workflows/deploy.yml` whenever `main` is updated.

One-time repo setting:

1. Open GitHub repository settings.
2. Go to **Pages**.
3. Set **Source** to **GitHub Actions**.

The custom domain is preserved via `public/CNAME`.
